# dotfiles

## Install

### Bash

```bash
git clone https://github.com/NathanVaughn/dotfiles.git
cd dotfiles/bash
chmod +x ./install
./install
source ~/.bash_profile
```

### PowerShell

```powershell
git clone https://github.com/NathanVaughn/dotfiles.git
cd dotfiles/pwsh
./install.ps1
. $PROFILE
```
